# SeenBoT
`An Advance BroadCasting Bot Based On `[`Telegram-Cli`](valtman.name/telegram-cli)

# Installation
- First Step :

  ```
  git clone https://github.com/SPR-CPU-ORG/SeenBot.git
  ```
- Second Step :

  ```
  cd SeenBot
  ```
- Third Step :

  ```
  chmod 777 Installer
  ```
- Forth Step :

  ```
  bash Installer
  ```
- End Step :
  
  ```
  lua MakeFile.lua
  #Enter Your ID And then "N"
  ```
# Running
  ``
    Just Use ./Runner ;)
  ``
